Инструменты веб-аналитики от Google

Google предлагает бесплатно следующие облачные технологии (нужна регистрация аккаунта Google):

  • сервис Google Analytics - перейти на сайт GA
  • тестовый аккаунт интернет-магазина Google - перейти на сайт GMS
  • справка по Google Analytics - перейти в справку GA
  • сервис Google Tag Manager - перейти в GTM
  • справка по Google Tag Manager - перейти в справку GTM
  • сервис визуализации данных Google Data Studio - перейти в GDS

Инструменты веб-аналитики от Яндекс

Яндекс предлагает бесплатно следующие облачные технологии:

  • сервис Яндекс.Метрика - перейти на сайт ЯМ
  • демо-аккаунт международного сайта Яндекс - перейти на сайт демо-счетчика
  • справка по Яндекс.Метрика - перейти в справку ЯМ
